Le blocus

Dans le cadre d'un projet en APL ( Algorithmique et Langages de Programmation), j'ai pu développer ce jeu avec un autre étudiant en utilisant le langage C et une bibliothèque graphique développer par un enseignant. Le but de ce jeu est de bloquer son adversaire, c'est un jeu en 2D qui prend la forme d'une grille de taille variable (de 3 à 9) choisi par le joueur. Le joueur doit faire deux actions pendant son tour :

  • Il doit poser une croix sur une case libre de la grille
  • Il doit ensuite bouger son pion sur une case adjacente (les pions peuvent bouger en diagonal)

La création de ce jeu m'a permis d'appliquer les principes vus en cours et en TP et aussi de développer mes compétences en terme de :

  • D'organisation du code
  • De compatibilité de fonction lors du développement à plusieurs personnes

Le jeu et le code source est disponible sur GitHub, pour que le jeu soit fonctionnel il faut installer la bibliothèque graphique de l'IUT.

Le SameGame

Le SameGame est un jeu codé en Java dans le cadre du cours d'APL pour mettre en application nos connaissances dans ce langage. Le jeu se joue tout seul, il y a une grille avec des blocs de différentes couleurs. Le but est d'avoir le score le plus élevé pour cela il faut : cliquer sur des groupes de blocs d'une même couleur pour pouvoir les enlever, plus le groupe de blocs est grand plus il rapporte de points.

On ne s'est pas attardé sur le graphisme, on a privilégié les fonctionnalités et corriger les bugs. Le jeu et le code source est disponible sur GitHub si vous voulez le tester.

Dico Anglais

J'ai fait ce programme pendant mon temps libre, il permet de rechercher la traduction d'un mot en une autre langue. Il n'a pas d'interface graphique, ce qui n'est pas forcément utile pour ce genre d'application. Quand on l'execute on est en "mode console" dans le programme comme dans gdb, sftp ou ssh. Il y a différentes commandes qui permettent de gérer les traductions:

  • add : ajoute une traduction
  • trad : traduit un mot en anglais ou ne français
  • help : donne la liste des commandes
  • del : supprime une traduction
  • exit : quitter l'application

Le code source est disponible sur GitHub.

Interface Homme Machine(IHM)

Ce projet a été fait dans le cadre du cours d'IHM (Interface Homme Machine). Il y a deux programmes un pour le gerant de l'hôtel et un autre pour avoir les statistiques

Le programme "Reservation" permet d'attribuer une chambre à un client qui a déjà fait sa résèrvation sur internet. Il permet aussi de choisir une autre chambre si celle-ci ne convient pas au client. La réservation peut être trouvée grâce au nom et prénom du client ou grace au numéro de réservation.

Le programme "statistique" permet de connaitre le taux de chambres occupées sur une journée ou sur plusieurs jours et le taux de non présentation (ceux qui ont réservé mais qui ne sont pas venus).

Le code source est disponible sur GitHub.
