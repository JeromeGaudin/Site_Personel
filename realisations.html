Le blocus

Dans le cadre d'un projet en APL ( Algorithmique et Langages de Programmation), j'ai pu développer ce jeu avec un autre étudiant en utilisant le langage C et une bibliothèque graphique développer par un enseignant. Le but de ce jeu est de bloquer son adversaire, c'est un jeu en 2D qui prend la forme d'une grille de taille variable (de 3 à 9) choisi par le joueur. Le joueur doit faire deux actions pendant son tour :

  • Il doit poser une croix sur une case libre de la grille
  • Il doit ensuite bouger son pion sur une case adjacente (les pions peuvent bouger en diagonal)

La création de ce jeu m'a permis d'appliquer les principes vus en cour et en TP et aussi de développer mes compétences en terme de :

  • D'organisation du code
  • De compatibilité de fonction lors du développement à plusieurs personnes

Le jeu et le code source est disponible sur GitHub, pour que le jeu soit fonctionnel il faut installer la bibliothèque graphique de l'IUT.
